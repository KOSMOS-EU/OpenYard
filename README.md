## KOSMOS - Kommunales Open Source Module Ökosystem

**KOSMOS** ist ein **migratives und nachhaltiges Ökosystem** für kommunale Software, das den Weg von proprietären Insellösungen hin zu einer integrierten, kosteneffizienten Open-Source-Cloud weist. OpenYard eröffnet das **DMS-Kapitel** dieser kommunalen Digitalisierungsrevolution.


*Entwickelt mit ❤️ und Verantwortung für nachhaltige kommunale Digitalisierung | KOSMOS - Von Kommunen, für Kommunen*
